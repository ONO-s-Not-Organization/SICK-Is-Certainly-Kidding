# 表格自由编辑优化方案

## 1. 交互设计：悬浮工具栏 (Floating Toolbar)
- **触发条件**: 当用户点击表格中的任意单元格（`input` 聚焦）时。
- **显示位置**: 紧贴当前单元格上方或左侧，避免遮挡内容。
- **功能按钮**:
  - `+↑`: 在上方插入行
  - `+↓`: 在下方插入行
  - `+←`: 在左侧插入列
  - `+→`: 在右侧插入列
  - `🗑️行`: 删除当前行
  - `🗑️列`: 删除当前列

## 2. 逻辑实现
- **插入逻辑**:
  - 使用 `Array.prototype.splice` 在指定索引位置插入数据。
  - 同步更新 `rowHeights` 和 `colWidths` 数组，保持尺寸对应关系。
- **删除逻辑**:
  - 同样使用 `splice` 或 `filter` 移除指定索引。
  - 增加安全检查，防止删除最后一行/列导致表格崩溃。

## 3. 视觉优化
- **高亮效果**: 
  - 为当前聚焦的单元格所在的整行和整列添加浅色背景高亮，类似于 Excel 的行号/列标高亮，方便用户确认操作范围。
- **过渡动画**: 按钮显示增加微弱的淡入效果，提升“成熟工具”的质感。

## 4. 进度安排
1. **第一步**: 重写 `TableEditor` 的增删逻辑函数。
2. **第二步**: 实现单元格聚焦状态管理。
3. **第三步**: 构建并定位悬浮工具栏 UI。
4. **第四步**: 添加行列高亮视觉反馈。
