# 诊断证明表格样式与布局优化方案

基于“逻辑而非单纯样式”的反馈，本方案将表格底层渲染机制改为 Flexbox，以支持更智能的单元格分布，并全面优化视觉效果使其符合医疗表单规范。

## 1. 核心布局优化 (`TableEditor.tsx`)
- **Flexbox 替代 Table 标签**：
  - 使用 `display: flex` 构建行和列。
  - **单元格自适应**：每个单元格设置 `flex: 1`。
  - **逻辑效果**：当您删除一个单元格时，该行剩余的单元格会自动拉伸以平分空间。这既满足了“该行减少单元格”的逻辑，又保证了表格整体矩形轮廓的完整，不会出现“缺角”。
- **医疗表单风格样式**：
  - **边框**：外边框加粗（如 2px），内边框保持细线（1px），符合打印表单的严肃感。
  - **文字对齐**：支持垂直水平居中，消除输入框自带的边距。

## 2. 交互与参考线优化 (`GuideLines.tsx` & `DraggableElement.tsx`)
- **参考线视觉升级**：
  - 颜色由红色改为淡蓝色 (`#3b82f6`) 或青色，减少视觉干扰，增加专业感。
  - 线条改为实线或更细的虚线。
- **吸附提示**：
  - 在元素被吸附时，除了参考线，增加轻微的边框颜色变化反馈。
